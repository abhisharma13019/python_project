# Python Practice Scripts
--------------------------------------------------

This repository contains simple Python programs created for learning and practice purposes.

--------------------------------------------------

## Files Included

- R_P_S.py        (Rock Paper Scissors Game)
- cal.py          (Calculator Program)
- cont_book       (Contact Book Program)
- pass_gen        (Password Generator)
- to-do.py        (To-Do List Application)
- README.md

--------------------------------------------------

## How to Run

Run any script using Python:

python3 file_name.py

Example:

python3 R_P_S.py
